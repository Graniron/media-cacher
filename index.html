Offline Media

Technical details of implementation This demo demonstrates media caching.
The caching has been implemented with Service Workers and Cache API.
The image/video source should use cross origin headers.
For testing purposes, images and videos can be used from Imgur.
To check caching use browser offline mode(or just turn off internet connection) and reload the page. Even without the internet you will be able to see content of the page.
